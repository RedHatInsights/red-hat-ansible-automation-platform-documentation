= Upgrading to {PlatformName} {PlatformVers}

:context: aap-upgrading-platform

[role="_abstract"]

To perform a basic upgrade of your {PlatformName} to a later version, download the desired version of the {PlatformNameShort} installer. Configure the `inventory` file with the same parameters you have used in previous installations and add new parameters to enable any new features, then run the installer.

include::platform/con-aap-upgrade-planning.adoc[leveloffset=+1]

Once you have completed the necessary upgrade planning, proceed by downloading the desired version of the {PlatformNameShort} installer.
