= Predefined content

:context: aap-azure-predefined-content

[role="_abstract"]
Red Hat provides predefined projects and job templates that you can deploy from {ControllerName} on an execution environment.
The preconfigured content enables you to automate the management of resources in your Azure subscription.

This chapter describes what the predefined content does, and how to view and use it, so that you can build similar automations.
