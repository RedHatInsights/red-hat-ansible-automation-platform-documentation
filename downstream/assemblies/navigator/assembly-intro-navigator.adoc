= Introduction to {Navigator}

:context: intro-ansible-navigator

[role="_abstract"]

As a content creator, you can use {Navigator} to develop Ansible playbooks, collections, and roles that are compatible with the {PlatformName}. You can use {Navigator} in the following environments, with seamless and predictable results across them all:

* Local development machines
* {ExecEnvNameStart}

{Navigator} also produces an artifact file you can use to help you develop your playbooks and troubleshoot problem areas.
